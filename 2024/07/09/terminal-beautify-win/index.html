使用 Visual Studio Code 时，还需要配置集成终端以使用 Nerd 字体。这可以通过更改终端设置中的值来完成（默认快捷方式：CTRL + ,）。选择Users，搜索Integrated: Font Family。

显示文件和文件夹图标
